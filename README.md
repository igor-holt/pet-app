# Dog Training AI App

A turnkey AI-powered dog training application. Ingests skill notecards, extracts data using Gemini 3 Pro, and generates personalized training plans.

## Features

- **Skill Ingestion**: Automatically extracts skill name, cue, and description from images using Gemini Vision.
- **Skill Browser**: Browse skills in a beautiful, responsive interface.
- **AI Training Plans**: Generate step-by-step positive reinforcement training plans for any skill using Gemini 3 Pro.
- **Turnkey Deployment**: Single container deployment with Docker.

## Prerequisites

- Docker & Docker Compose
- A Google Gemini API Key

## Quick Start

1.  **Add Data**: Place your dog skill notecard images (JPG/PNG) in `data/dog_skill_notecards/`.
2.  **Set API Key**: Export your Gemini API key:
    ```bash
    export GEMINI_API_KEY="your_api_key_here"
    ```
3.  **Run**:
    ```bash
    docker compose up --build
    ```
4.  **Access**: Open [http://localhost:8000](http://localhost:8000).
5.  **Ingest**: Click the "Refresh Data" button on the home page to trigger the ingestion process.

## Deployment

To deploy to a cloud provider (e.g., Google Cloud Run):

1.  **Build the container**:
    ```bash
    docker build -t dog-training-app .
    ```
2.  **Push** to your container registry.
3.  **Deploy** passing the `GEMINI_API_KEY` environment variable.
4.  **Data Persistence**: Mount a volume to `/app/data` to persist the SQLite database and access images, or bake the images into the image if they are static.

## Development

- **Backend**: FastAPI (Python) in `backend/`
- **Frontend**: React + Vite in `frontend/`
- **Database**: SQLite in `data/skills.db`

## Safety Note

All training plans are generated by AI. Always consult a professional dog trainer for behavioral issues or aggression.
